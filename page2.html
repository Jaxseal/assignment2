CSS Box Model

What is The CSS Box Model

The box model is a model of all of the around an element which is the content its self, the padding, the border and the margin. As you can see there is a diagram of this on webpage as you can see the content is where the blue box is, the padding is where the green box is, the border is where the yellow box is and the margin is the last layer which is the orange layer.
